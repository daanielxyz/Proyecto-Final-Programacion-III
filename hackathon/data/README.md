# Carpeta de Datos

Esta carpeta contiene los archivos JSON donde se persiste la información de la Hackathon.

## Archivos:
- `equipos.json` - Lista de equipos registrados
- `participantes.json` - Lista de participantes
- `proyectos.json` - Proyectos de los equipos
- `mensajes.json` - Historial de mensajes del chat
- `mentores.json` - Mentores registrados

**NOTA:** Estos archivos se generan automáticamente al usar la aplicación.
